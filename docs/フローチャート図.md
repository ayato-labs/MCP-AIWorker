# Sub-cheap-McpAiAgent フローチャート

```mermaid
graph TD
    %% スタイル定義
    classDef main fill:#d4edda,stroke:#28a745,stroke-width:2px;
    classDef mcp fill:#cce5ff,stroke:#004085,stroke-width:2px;
    classDef llm fill:#fff3cd,stroke:#856404,stroke-width:2px;
    classDef file fill:#f8d7da,stroke:#721c24,stroke-width:2px;

    %% 処理フロー
    Start([1. メインAIがツールを呼び出す]) --> |引数: 指示, パス, 開始/終了行, 参考情報| Init[2. MCPサーバーが引数を受理]
    class Start main;
    class Init mcp;

    %% ツール分岐
    Init --> CheckTool{3. どのツールを使う?}
    class CheckTool mcp;
    
    CheckTool -->|draft_edit| FileCheck{ファイル存在判定}
    CheckTool -->|generate_unit_tests| UnitTestPipeline[ユニットテスト生成パイプライン]
    CheckTool -->|execute_command| LogSummarize[ログ要約パイプライン]
    CheckTool -->|fetch_and_summarize_url| WebSummarize[Web要約パイプライン]

    %% ユニットテストパイプライン
    UnitTestPipeline --> PlanGeneration[LLM: テスト計画(XML) + コード生成]
    PlanGeneration --> ExtractArtifacts[XMLタグ解析: <test_plan>, <draft_output>]
    ExtractArtifacts --> WriteTestFile[テストファイル生成]
    
    class UnitTestPipeline,PlanGeneration,ExtractArtifacts,WriteTestFile mcp;
    CheckLine -->|Yes| SetSnippetPart[7. 指定範囲の行だけを切り出す]
    CheckLine -->|No| SetSnippetFull2[8. ファイル全体のコードを読み込む]
    class SetSnippetPart,SetSnippetFull2 mcp;

    %% 合流とLLM呼び出し
    SetSnippetFull --> BuildPrompt[9. サブLLM用の軽量プロンプトを作成]
    SetSnippetPart --> BuildPrompt
    SetSnippetFull2 --> BuildPrompt
    class BuildPrompt mcp;

    BuildPrompt --> CallSubLLM[10. 安価なLLMを呼び出し]
    class CallSubLLM llm;

    CallSubLLM --> ReceiveCode[11. コードのみの出力を受理・トリミング]
    class ReceiveCode mcp;

    ReceiveCode --> WriteFile{12. 書き込みモードの判定}
    class WriteFile mcp;

    %% 書き込み処理
    WriteFile -->|新規 / 全体上書き| DirectWrite[13. ファイルにそのまま書き込み]
    WriteFile -->|特定行指定| LineMerge[14. 指定行の範囲だけを置換して書き込み]
    class DirectWrite,LineMerge file;

    %% 終了
    DirectWrite --> Response[15. メインAIに完了通知のみを返す]
    LineMerge --> Response
    Response --> End([16. 処理終了])
    class Response main;
    class End main;
```
